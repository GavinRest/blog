OpenList 安装 ​

  • 安装
docker run -d --restart=unless-stopped -v /etc/openlist:/opt/openlist/data -p 5244:5244 -e PUID=0 -e PGID=0 -e UMASK=022 --name="openlist" openlistteam/openlist:latest

  • 访问：http://192.168.5.10:5244/ 进入 web 控制台界面

  • 添加存储

http://192.168.5.10:5244/@manage/storages
添加存储 > 夸克 > 填写 cookies
  • 新建用户开通 webdav 权限
http://192.168.5.10:5244/@manage/users
添加用户 > 配置用户名密码 > 开通 webdav 权限
